# Creating the GitHub Issue

The Vite migration evaluation has been documented in `VITE_MIGRATION.md` and pushed to branch `claude/evaluate-vite-migration-gW0J7`.

## Option 1: Create Issue via GitHub Web UI

1. Go to https://github.com/cloverich/chronicles/issues/new
2. Title: **Evaluate and migrate renderer to Vite**
3. Copy the contents from `VITE_MIGRATION.md` into the issue body
4. Add labels: `enhancement`, `infrastructure`
5. Submit

## Option 2: Use GitHub CLI (if installed)

```bash
gh issue create \
  --title "Evaluate and migrate renderer to Vite" \
  --body-file VITE_MIGRATION.md \
  --label enhancement,infrastructure
```

## Option 3: Use API Script

If you have a GitHub token, run:

```bash
export GITHUB_TOKEN=your_github_token
/tmp/create_issue.sh
```
